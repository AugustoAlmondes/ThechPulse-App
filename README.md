## 📖 Descrição do Projeto

**TechPulse** é um aplicativo mobile focado em trazer as últimas notícias e novidades do mundo da tecnologia. Ele foi projetado para oferecer uma experiência de leitura rápida, intuitiva e agradável, permitindo aos usuários estarem sempre atualizados. Com uma interface moderna, o aplicativo permite salvar notícias favoritas, gerenciar o que ler mais tarde e acompanhar o histórico de acessos.

## ✨ Funcionalidades e Demonstração da Aplicação

O aplicativo conta com diversas funcionalidades pensadas para a melhor experiência do usuário:

- **Feed de Notícias:** Visualize as notícias mais recentes do universo da tecnologia, em múltiplos idiomas.
- **Leitura de Notícias:** Leia o conteúdo completo de forma imersiva e sem distrações.
- **Favoritos:** Salve suas notícias preferidas para acesso rápido a qualquer momento.
- **Ler Mais Tarde:** Adicione artigos interessantes em uma lista dedicada para ler depois.
- **Histórico:** Acompanhe os artigos e notícias que você já acessou.
- **Compartilhamento:** Compartilhe notícias interessantes diretamente do aplicativo.

## 🚀 Acesso ao Projeto

Siga os passos abaixo para baixar, instalar e rodar o projeto na sua máquina local.

### Pré-requisitos
- [Node.js](https://nodejs.org/en/)
- [Git](https://git-scm.com/)
- Conta no [Expo Go](https://expo.dev/go) (Para rodar direto no dispositivo físico) ou um emulador configurado.

### Passos para execução

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/TechPulse-App.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd TechPulse-App
   ```

3. Instale as dependências:
   ```bash
   npm install
   ```

4. Inicie o servidor do Expo:
   ```bash
   npx expo start
   ```

5. Abra o aplicativo através do **Expo Go** no seu celular ou utilize um **emulador** (Android/iOS).

## 🛠 Tecnologias Utilizadas

As seguintes ferramentas e tecnologias foram utilizadas na construção do projeto:

- **[React Native](https://reactnative.dev/)**: Framework para desenvolvimento do app mobile.
- **[Expo](https://expo.dev/)**: Plataforma para facilitar o desenvolvimento e testes em React Native.
- **[TypeScript](https://www.typescriptlang.org/)**: Superset do JavaScript que adiciona tipagem estática.
- **[Zustand](https://github.com/pmndrs/zustand)**: Gerenciamento de estado global da aplicação.
- **[React Navigation](https://reactnavigation.org/)**: Gerenciamento de rotas e navegação.
- **[Axios](https://axios-http.com/)**: Cliente HTTP para consumo de APIs.
- **[React Query](https://tanstack.com/query/latest)**: Gerenciamento de cache e requisições de dados assíncronos.
- **[Async Storage](https://react-native-async-storage.github.io/async-storage/)**: Armazenamento de dados localmente no dispositivo.
